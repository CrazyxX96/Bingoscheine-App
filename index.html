🤖 KI-Erkennung
📖 So richtest du deinen eigenen Key ein:
1. Auf console.anthropic.com ein kostenloses Konto erstellen
2. Unter Billing Guthaben aufladen (5 $ reichen sehr lange)
3. Unter API Keys → "Create Key" einen Key erstellen
4. Den Key (beginnt mit sk-ant-) oben eintragen und speichern
🔒 Dein Key bleibt nur auf diesem Gerät gespeichert — er wird nicht in die Cloud synchronisiert und ist für niemanden sonst sichtbar. Pro Scan fallen wenige Cent oder weniger an, abgerechnet über dein eigenes Anthropic-Guthaben.
💾 Datenspeicherung
Deine Daten werden lokal auf diesem Gerät im Browser-Speicher (localStorage) gesichert.
☁️ Mit aktivierter Cloud-Synchronisation werden sie zusätzlich unter einem anonymen Code online gesichert — so hast du auf allen Geräten dieselben Scheine, Zahlen und Finanzen und kannst sie nach einem Gerätewechsel wiederherstellen.
⚠️ Ohne Cloud-Anmeldung gilt: Wenn du den Browser-Cache oder die Website-Daten löschst, gehen alle gespeicherten Daten unwiderruflich verloren. Mit Anmeldung genügt danach eine erneute Anmeldung mit derselben E-Mail.
